# Aplikasi Web Tata Cara Sholat Muhammadiyah

## Deskripsi
Aplikasi web ini dibuat untuk memberikan panduan tata cara sholat sesuai Himpunan Putusan Tarjih Muhammadiyah.

## Fitur
- Tata cara Wudhu
- Tata cara Sholat
- Bacaan Sholat
- Audio Bacaan
- Jadwal Sholat

## Teknologi
- HTML
- CSS
- JavaScript
- MySQL
